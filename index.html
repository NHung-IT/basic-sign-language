🎯 Team Project

Basic Sign Language Recognition using Teachable Machine

🔍 Project Goal

Our goal is to enable computers to recognize basic sign language gestures using machine learning. This helps raise awareness about communication barriers for the hearing-impaired and supports daily interactions in education and public services. The project promotes inclusivity and shows how beginner-friendly tools like Teachable Machine can be applied to real-world problems.

📁 Dataset & Class Configuration

  • Collected hand gesture images for numbers 0–5
  • Each class includes 900–1000 images
  • Diverse lighting, backgrounds, and hand positions
  • Clear, logically organized class labels

🛠️ Teachable Machine Setup

  • Used Image Project > Standard Image Model
  • Trained the model with our dataset
  • Split data into training and testing sets

💻 Code & Customization

  • Exported the model as TensorFlow code
  • Modified Python script for webcam & image input
  • Interface shows prediction confidence and feedback

🚀 Running the Project

  • Real-time gesture recognition via webcam
  • Option to upload images for recognition
  • Shows prediction confidence percentage

🌱 Future Plans

  • Expand gesture recognition to full alphabet and common phrases
  • Integrate voice output for real-time interaction
  • Develop a mobile version using TensorFlow Lite
  • Improve dataset with more users and environments
  • Create a tool that supports real hearing-impaired users
